## Versioning
Heroic Tales uses semantic versioning to indicate changes to the Core Rules. Each version is given a three-part number in the format MAJOR.MINOR.PATCH, where each number is incremented according to the following:

1. MAJOR version when changes are made to the core mechanics that break backwards compatibility with previous versions,
2. MINOR version when new rules and content are added that maintain compatibility with the previous versions,
3. PATCH version when typos, design issues, ambiguous wording, or other minor corrections are addressed.
