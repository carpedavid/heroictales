I’m proud to announce that Heroic Tales, the genre-neutral role-playing game of heroic problem solving, has reached a new milestone: 2.0.0. As I was writing an upcoming solo supplement for Heroic Tales, I made some significant changes to the underlying core rules. Among them:

- Simplified character creation
- Simplified the task resolution mechanics AND added bounded accuracy, so the same task will now be easier as your character advances in level
- Simplified the character advancement rules
- Added a new optional framework for describing character personality
- Added lightweight rules for currency
- Completed a huge rewrite of the “Playing the game” section to read better, reduce redundancy, and be easier to learn.

The Heroic Tales core rules and character sheet are available FOR FREE from:

- [Heroic Tales website](/publications/core-rules/)
- [DriveThruRPG](https://www.drivethrurpg.com/product/346856/Heroic-Tales-Core-Rules)
